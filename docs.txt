Компоненты для построения приложения берутся из react-native, они универсальны, под капотом уже реакт натив сам разбирает какой платформе какие компоненты подправить.
http://facebook.github.io/react-native/ - официальный сайт реакт нэйтив

Мы не можем использовать теги html только РН компоненты
Текстовый контент должен быть заключен в теги <Text>
Блоки для верстки типа Div в РН называются <View>

Стили пишутся инлайн на js в объектах, все свойства из 2 слов записываются в
кэмэл кейс формате font-size => fontSize и некоторые принимают парапметрами строку
а некоторые только число:
flex: 1 - сделать блок занимающим всю высоту, по умолчанию все элементы являются
display: flex;

Страницами в мобилках принято называть скрины screens
