Мы не можем использовать теги html только РН компоненты
Текстовый контент должен быть заключен в теги <Text>
Блоки для верстки типа Div в РН называются <View>

Стили пишутся инлайн на js в объектах, все свойства из 2 слов записываются в

Работа с контекстом
Контекст работает как оболочка над основным компонентом приложения. В нем будет храниться состояние
и производиться работа с ним.
1 - создаем файл для работы с контекстом todoContext.js
  импортируем туда createContext метод из пакета react, далее экспортируем
  результат работы функции createContext и им мы будем пользоваться.
2 - создаем промежуточный компонент TodoState.js который будет создавать
  обёртку контекста, в нем будем делать все логические операции и его делаем
  как обертку. Импортируем react и наш созданный контекст todoContext.js
  Далее экспортируем наш стейт компонент TodoState который будет стрелочной
  функцией принимающей в себя параметр children и возвращающий jsx компонент
  TodoContext и его сущность Provider. Принимает компонент в себя value объект
  куда мы экспортируем наши переменные методы все что запихнем в контекст.
3 - для хранения стейта в TodoState импортируем из пакета react метод useReduser
  который заменит метод useState. Первый параметр будет ф-ция редюсер, вторым
  передается начальное состояние. Вытягиваем из результата работы метода
  useReduser нужные нам вещи: state и dispatch
4 - создаем редюсер. Экспортируем ф-цию редюсер, которая принимает в себя стейт
  и экшин с которым мы будем взаимодействовать. Возвращает редюсер state.
5 - ипортируем наш редюсер в TodoState
